# CarRental
Optimized time complexity and performance by refactoring program in NodeList and Iterator-based implementation and enhanced it by exceptions, allowing functions to reserve/return a car, check availability and print action history updates in a log.   

# Function Highlight
-PrintList(): print a list of all vehicles
-PrintAvailableList(): print out a list of all available vehicles using seats
-CheckAvailability(): check if a car is reserved or not by entering the ID
-Reserve() and Return(): reserve or return a vehicle
-PrintAvailableList(): print out a list of all available vehicles using seats
